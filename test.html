ZeroClipboard for jQuery

Hello! My name is Kris, I created (Or, modified) this script to allow ZeroClipboard to mingle with jQuery a bit better. The main goal of this project was to make a very simple clipboard feature, as the original didn't meet my needs. The problem was that I had a site with multiple clipboards, needing to be managed and updated by the elements, and so this version stores all the data in the elements, and pairs the element and flash object with an array. Also, its like any other jQuery plugin you have seen :) Below are some examples on the use, and shows off some of its abilities.

Important Notice! This scripts resize abilites are provided by a very helpful script over at BelAlman.com. It takes the resize functionality of window, and applies it to other elements. Grab a copy from this blog post to enable the feature in this script. The only other requirement is jQuery, but we all knew that one :)

Binding Events

Finally, we come to the binding of events. There are 4 types of events. mouseOver, mouseOut, mouseDown, mouseUp. Using events is also simple:

Watch me change colors!

This is done with events. With this the CSS effects can be much more specific, and you can call other events at certain stages. The this in the element refers to the element the flash object is paired to.

The following is an example in using the events, its just as you would with any event on the div.

$('#testercss').zeroclipboard({
  'text': 'Copied from #testercss',
  'hand': true
}).mouseover(function () {
  $(this).css({'color': '#f00'}, 1000);
}).mouseout(function () {
  $(this).css({'color': '#000'}, 1000);
}).mousedown(function () {
  $(this).css('font-weight', 'bold');
}).mouseup(function () {
  $(this).css('font-weight', 'normal');
});
